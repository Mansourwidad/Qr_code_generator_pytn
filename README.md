# 📱 QR Code Generator

A simple Python tool to generate QR codes from text or URLs with custom colors.

## ✨ Features

- Convert any text or URL to QR code
- Custom colors (fill and background)
- Save as PNG image
- Simple command-line interface

## 🚀 Installation

1. Make sure you have Python installed (3.6+)

2. Install required packages:
```brash
pip install -r requirements.txt
